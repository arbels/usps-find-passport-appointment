# Usage

From the root directory run:

`yarn`

`yarn start`

## Configuration

Before running the app for the first time, create a `.env` file in the root folder holding the config variables necessary for the app to send alerts.

```
MAX_DAYS=
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE_NUMBER=
MY_NUMBER=
```
